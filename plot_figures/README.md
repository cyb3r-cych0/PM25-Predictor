## ÔøΩÔøΩ Usage

- **Viewing Plots**: Open PNG files directly to inspect visualizations.
- **Trend Analysis**: Use individual trend plots to evaluate how well each model captures long-term PM‚ÇÇ.‚ÇÖ trends.
- **Comparisons**: Review combined and supplementary plots for multi-model diagnostics.

## üß† Integration with Pipeline

- Generated by `../plot_scripts/models_actual_vs_pred_trends_plots.py`, `../plot_scripts/models_pred_vs_obs_plots.py` and
 `../plot_scripts/models_bias_trend_timeseries.py`.
- Consumes data from `../models_pipeline_data/` (predictions, trends).
- Used in reporting and analysis to support model evaluation.

Ensure plotting scripts are run after model training and trend computation to populate this directory.
